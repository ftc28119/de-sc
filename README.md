# FTC Scout 数据管理系统 - 使用须知

## 项目介绍

此WEB是#28119为First Tech Challenge Season 2025-2026 DECODE设计的scouting APP

## 快速开始

### 1. 注册与登录

1. 访问前端页面：https://28119scout.up.railway.app
2. 点击"注册"按钮，填写用户名、密码和队伍编号
3. 注册成功后自动登录
4. 或直接使用已有的账号登录

### 2. 加入或创建队伍

- **创建队伍**：注册时填写队伍编号，系统自动创建队伍
- **加入队伍**：使用队长提供的邀请码注册，或注册后联系队长邀请

### 3. 录入比赛数据

1. 登录系统
2. 填写基本信息：队伍编号、比赛名称、比赛类型、比赛编号
3. 选择Motif：GPP、PGP或PPG
4. 填写自动阶段数据：
   - 机器人驶离发射线
   - 溢出GATE的Artifacts数量
   - 溢出GATE并滚落至球道的Artifacts数量
   - 自动阶段结束时的分类情况
5. 填写手动阶段数据：
   - 仓库线得分
   - 溢出GATE的Artifacts数量
   - 溢出GATE并滚落至球道的Artifacts数量
   - 基地返回状态
   - 手动阶段结束时的分类情况
6. 填写通用信息：
   - 驾驶表现评分
   - 防御评级
   - 机器人故障情况
   - 投篮命中情况
   - 备注
7. 点击"提交数据"按钮保存

### 4. 查看数据

1. 登录系统
2. 点击"查看数据"按钮
3. 在搜索框中输入要查询的队伍编号
4. 查看数据列表，点击"查看详细"查看完整数据
5. 点击"删除"删除不需要的数据

### 5. 导入/导出数据

- **导出数据**：点击"导出数据"按钮，将数据保存为JSON文件
- **导入数据**：点击"导入数据"按钮，选择JSON文件导入

## 功能说明

### 用户管理

- **注册**：创建新账号，可选择创建新队伍或加入现有队伍
- **登录**：使用用户名和密码登录系统
- **注销**：退出当前账号
- **修改密码**：在个人中心修改密码
- **删除账号**：在个人中心删除账号

### 队伍管理

- **创建队伍**：注册时自动创建队伍
- **邀请成员**：队长可生成邀请码，用于邀请其他成员加入
- **刷新邀请码**：队长可刷新邀请码，使旧邀请码失效
- **查看成员**：在队伍详情中查看所有成员

### 比赛数据管理

- **录入数据**：详细记录比赛各阶段的数据
- **查看数据**：只能查看自己队伍记录的数据
- **搜索数据**：根据队伍编号搜索数据
- **删除数据**：删除不需要的数据
- **导入/导出**：支持数据的导入和导出

### 数据分析

- **实时得分计算**：自动计算当前录入数据的得分
- **数据统计**：统计比赛数据，生成报告

## 注意事项

### 数据安全

1. **数据权限**：每个队伍只能查看自己队伍记录的数据
2. **账号安全**：
   - 请勿泄露账号密码
   - 定期修改密码
   - 公共电脑使用后请及时注销
3. **数据备份**：
   - 定期导出数据进行备份
   - 防止数据丢失

### 使用规范

1. **数据录入规范**：
   - 确保数据真实、准确
   - 及时录入比赛数据
   - 填写完整的比赛信息
2. **队伍协作**：
   - 明确分工，避免重复录入
   - 保持数据格式统一
3. **系统使用**：
   - 不要同时在多个设备登录同一账号
   - 系统可能会自动保存数据，但建议手动保存
   - 定期清理无用数据

## 故障排除

### 1. 无法登录

- 检查用户名和密码是否正确
- 确保网络连接正常
- 尝试重置密码

### 2. 数据无法提交

- 检查网络连接
- 确保所有必填字段已填写
- 检查数据格式是否正确

### 3. 无法查看数据

- 确保已登录账号
- 确保该账号已加入队伍
- 检查是否有数据权限

### 4. 数据显示异常

- 刷新页面
- 清除浏览器缓存
- 检查网络连接
